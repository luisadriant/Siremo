Que es Siremo

Es un sistema de recomendacion de moda y estilo que permite a los usuarios idenficar y ver las recomendaciones de diferentes prendas de vestir que se encuentran en el mercado, permitiendo elegir en gustos, tamaño, precios, etc. Cuenta con un algoritmo de recomendacion, el cual se basa en analizar y procesar información histórica de los usuarios (edad, género, calificaciones..), de las vestimentas (marcas, modelos, precios..) y realiza la predicción de los productos que pueden ser interesantes para el usuario.

Como funciona Siremo

Siremo es un sistema de recomendación que utiliza un sistema experto basado en reglas para definir el estilo de cada uno de los usuarios, ademas este sistema provee al usuario recomendaciones de prendas de vestir, basandose en predicciones exactas ya que utiliza los algoritmos "Fuzzy C Means" y "K-Vecinos".

Como utilizarlo

Para poder utilizar SIREMO solo es necesario registrarte con tus datos personales y llenar el test que aparece en el primer inicio de sesion y podras obervar toda la vestimenta disponible y de este modo podras disfrutar de los beneficios que brinda este sistema para tu forma de vestir.
